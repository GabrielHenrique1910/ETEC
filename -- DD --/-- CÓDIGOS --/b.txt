3. **Lógica de Morte e Efeitos:**

   No evento **Step** do inimigo, você pode adicionar código para executar ações quando a vida (`hp`) chega a zero. Aqui está um exemplo completo, incluindo efeitos visuais e sonoros:

   ```gml

   if (hp <= 0) {
       // Executa a animação de morte ou outros efeitos

           instance_create_layer(x, y, "Effects", obj_death_effect);
       }
       
       // Adiciona um efeito sonoro de morte

           audio_play_sound(snd_death, 1, false);
       }
       
       // Destrói o inimigo
       instance_destroy();
   }
   ```

   - **obj_death_effect**: Um objeto que você pode criar para mostrar uma animação ou efeito quando o inimigo morre.
   - **snd_death**: Um som de morte que será tocado quando o inimigo for destruído.

4. **Adicionar Feedback Visual ou Sonoro:**
